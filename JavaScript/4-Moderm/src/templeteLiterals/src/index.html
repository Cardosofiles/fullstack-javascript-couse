Exibindo o template literal

O template literals em JavaScript são uma maneira de criar strings mais dinâmicas e legíveis, permitindo a incorporação de expressões JavaScript e a formatação multi-linha diretamente dentro da string. Eles são definidos usando a sintaxe de crase (` `) em vez de aspas simples ou duplas.
